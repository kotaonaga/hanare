# はなれ


「ため息をつくと幸せが逃げる。」こんな言葉を聞いたことがあるだろう。しかし、時にはため息をつく必要がある。
うまくため息をつく。その瞬間に救われることもある。ため息をつく時間を大切にしたい。
そんな思いからこの装置は生まれた。ため息は風に変換され風鈴を揺らす。
自ら音を奏でることを放棄した風鈴をはじめとしたそれぞれの物体が役割を放棄し、一息ついている。  

ProcessingとArduinoを用いて制作。
